Pyvelox Arrow Api¶

pyvelox.arrow.to_velox(arg0: object) → pyvelox.vector.Vector¶

Converts an arrow object to a velox vector.

Parameters:

vector – Input arrow object.

Examples:

>>> array = pyarrow.array([1, 2, 3, 4, 5, 6])
>>> vector = to_velox(array)
pyvelox.arrow.to_arrow(arg0: pyvelox.vector.Vector) → object¶

Converts a velox vector to an arrow object.

Parameters:

vector – Input arrow object.

Examples:

>>> import pyvelox.legacy as pv
>>> vec = pv.from_list([1, 2, 3, 4, 5])
>>> arrow = to_arrow(vec)
